# TaskbarTimer

Windows 任务栏托盘计时器 - 简洁的桌面秒表工具。

## 功能

- ⏱️ 秒表计时（开始/暂停/清空）
- 📍 悬浮任务栏上方，可拖动
- 🎨 颜色状态提示（绿色计时中/黄色暂停/蓝色未开始）
- 🕐 实时显示系统时间

## 使用方法

### 方式一：双击运行（推荐）
双击 `启动计时器.bat` 即可启动。

### 方式二：手动运行
右键 `TaskbarTimer.ps1` → 选择 **"使用 PowerShell 运行"**。

## 系统要求

- Windows 7 / 8 / 10 / 11
- PowerShell（系统自带）

## 退出方式

右键任务栏托盘区域的时钟图标，选择"退出"。
